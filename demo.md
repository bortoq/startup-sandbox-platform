# Demo

## Example Input

```
A platform for small household inventions: people publish useful improvements,
validate demand, prototype them, and sell or license micro-batches.
```

## Generated Project Space

### Project Brief

The system creates a short description, target users, problem, solution, MVP, risks, and monetization notes.

### Roadmap

The system generates phases:

- idea format;
- validation;
- prototype;
- micro-batch;
- sales;
- licensing.

### Landing Page

The system drafts a public page with a one-line value proposition, call to action, and early interest form.

### Validation Checklist

- Who has this problem?
- What do they use now?
- Would they pay?
- Can the product be made cheaply?
- What is the smallest test?

### Funding And Tasks

The system creates pledge goals, task cards, contributor roles, and first experiments.

## Demo Goal

The first demo should show raw idea to public project page in one flow.
